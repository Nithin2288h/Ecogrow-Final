# Default Product Images

This directory contains default product images organized by category.

## Directory Structure

```
default-images/
├── vegetables/
├── fruits/
├── millets/
└── grains/
```

## Image Naming Convention

Images should be named using the product name in lowercase with hyphens, e.g.:
- `tomato.jpg`
- `bitter-gourd.jpg`
- `green-peas.jpg`

If images are not available, the application will automatically generate placeholder images with the product name.

## Supported Products

### Vegetables
- Tomato, Potato, Brinjal, Carrot, Onion, Cabbage, Cauliflower, Capsicum, Spinach, Green Peas, Bottle Gourd, Bitter Gourd, Radish, Beetroot

### Fruits
- Apple, Banana, Mango, Orange, Papaya, Pineapple, Grapes, Watermelon, Pomegranate, Strawberry, Guava, Coconut

### Millets
- Ragi, Bajra, Jowar, Little Millet, Kodo Millet, Foxtail Millet, Barnyard Millet

### Grains
- Rice, Wheat, Maize, Barley, Oats, Sorghum
